# truST plugins for Claude Code

Plugins that bring the [truST](https://github.com/johannesPettersson80/trust-platform) IEC 61131-3 toolchain into Claude Code.

## Install

```
/plugin marketplace add johannesPettersson80/trust-claude-plugins
/plugin install trust-st-lsp@trust-claude-plugins
```

## Plugins

- **trust-st-lsp** — language server for IEC 61131-3 Structured Text. Diagnostics, jump-to-definition, find references, hover, document/workspace symbols, and call hierarchy on `.st` files.
